## What you're doing and why

You've learned what agents are, how to use Claude from the terminal, and how tools extend AI capabilities. Now you're going to build something real.

"Build" here doesn't mean writing code. It means designing a task for Claude Code and running it — letting the agent loop handle the execution.

## Prerequisites

- Claude Code installed (`npm install -g @anthropic-ai/claude-code`)
- An Anthropic API key set in your environment
- A folder with real content to work with

## Choose your task

Pick one of these, or design your own:

**Option A — Document summariser**
Point Claude at a folder of documents (PDFs, text files, notes) and ask: "Read all the files here and create a single summary document that captures the key points from each one."

**Option B — Project explorer**
In a code repository or project folder, ask: "Read this project and create a OVERVIEW.md file that explains what this project does, its structure, and how to get started."

**Option C — Meeting notes processor**
Paste or save your last meeting notes as a text file, then ask: "Read this meeting transcript and create an action items list with owners, a summary for people who weren't there, and a follow-up email draft."

**Option D — Your own task**
What do you do repeatedly that involves reading, writing, or organising information? Design a task for Claude Code to handle.

## Running it

```bash
cd /path/to/your-folder
claude
```

Then describe your task clearly: what the input is, what you want as output, and any constraints.

Watch what happens. The agent will:
1. Read the relevant files
2. Process the content
3. Write output files or display results
4. Ask if it needs clarification

## What to notice

- Did it interpret your task correctly?
- Where did it take an unexpected approach?
- What would you do differently in the prompt next time?

## Key takeaway

You've now built with AI — not used it. You gave a goal to an agent, watched it reason and act, and got a real output. That's the shift this module is designed to create.
